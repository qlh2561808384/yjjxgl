初始化参数

名称类型描述默认值
widthintinput宽度100%
selectedint默认选中 参数为 option index 由于0表非 从1开始计0
defaultValuestring根据匹配的 option value 进行默认选中""
requiredBoolean是否必选false
validateFunction OR String 定义表单验证规则。可使用globalValid中定义的正则名称 支持多种正则验证 多个 正则名间用'|'进行分割。如正则无法满足 可定义function(value){return {result:false,msg:''}} 参数为表单value 返回值 包含boolean 标识验证是否成功 当result为false时 提示msg 错误信息false
validTextString自定义验证提示文本""
validFailFunction验证失败触发回调方法 参数为 fn jquery对象function($el){return false}
searchOptionbooleantrue时 添加一个value为'ALL'的'全部'选项 false时 添加一个value为''的空选项null
valueFieldString定义取值字段名'id'
textFieldString定义显示文本 字段名'text'
localdata[]本地数据 用于加载本地数据[]
urlString远程数据请求路径''
queryParamobj远程请求参数对象 例：{param1:123,param2:456}{}
panelWidthint下拉面板宽度auto
panelHeightint下拉面板高度auto
multipleboolean是否支持多选false
separatorString多选状态下数值和文本间的分隔符符号','
hasDownArrowboolean是否显示下拉按钮true
onShowPanelFunction下拉面板展开时的回调方法 return false
onHidePanelFunction下拉面板隐藏时的回调方法return false
onChangeFunctionchange触发回调方法 参数1：newValue参数2：oldValuereturn false
onClickFunction点击时触发回调方法return false

方法:

名称参数描述
destroynone销毁表单组件
validatenone对input进行表单验证 返回boolean
clearnone清除文本内容
getValuenone获取value
setValueString设值回写 多选模式下 逗号进行分隔
disablenone禁用组件
showPanelnone显示下拉面板
hidePanelnone隐藏下拉面板
getTextnone获取显示文本内容
setTextString设置显示文本内容
reloadobject重新加载选项数据 参数为 请求对象：{url:'',queryParam:{}} 或 本地数据对象:[{},{}]
