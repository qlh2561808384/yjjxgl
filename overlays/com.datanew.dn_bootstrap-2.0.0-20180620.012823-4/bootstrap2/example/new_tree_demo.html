属性：

名称类型描述默认值
searchPlaceholderstring查询栏查询字符串占位符
outerHeightNumber固定树的高度，超出部分滚动undefined
fixedTreeHeightNumber有查询框的情况下，树的高度
（为了让输入框一直显示在顶部，需要固定树的高度，超出部分滚动）		(body的高度)减去
(输入框高度)的值
localdataArray本地数据，zNodes =[{id:1, pId:0, name:"节点搜索演示 1"}]
urlString远程数据请求地址
queryParamobj远程请求所携带的参数 格式：{param1:xxx,param2:xx}
openSearchboolean是否显示搜索框 开启搜索栏后 根据data.key.name 进行模糊查询 回车触发查询方法false
checkTypeString定义选择框类型，checkbox为多选框，radio为单选框"radio"
ISLEAFBoolean当checkType为checkbox或radio时，父节点是否有选择框，设置为false表示父不可选false
rootElementBoolean是否添加“全部”节点false
onLoadedfunction树加载完毕后 所执行方法 包含一个参数 ztree对象 可使用原生api 实现展开所有节点 等操作false
data 目前加载的数据格式均使用 simpleData 模式 即 简单的数组数据 具体具体数据格式样例可右键查看源代码

节点属性：

名称类型描述
colorString节点字体颜色 同 css 属性

方法:

名称参数描述
getTreenone返回ztree对象 用于调用ztree api 语法为：ztree对象.方法名。eg：$("#tree").tree("getTree");
getValuenone返回选择节点的value值 多选模式下逗号分隔。eg：$("#tree").tree("getValue");
setValuenone设置节点的value值 多选模式下逗号分隔。eg：$("#tree").tree("setValue", '1,2,3');
getTextnone返回选择节点的文本内容 多选模式下逗号分隔。eg：$("#tree").tree("getText");
clearTreenone清空选项。eg：$("#tree").tree("clearTree");
reloadobj重新加载内容 dataOption url参数 或本地数据对象 eg：$("#tree").tree("reload", {localdata:[]}); OR $("#tree").tree("reload", {url:XXXX,queryParam:{}});
searchNodesstring根据参数 模糊匹配节点name 隐藏非匹配节点 输入参数为''时 显示所有节点
ztree一些常用方法:
expandAllboolean(true)展开/(false)折叠所有节点 本案例中onLoaded 回调使用该方法实现展开树所有节点。eg：ztree.expandAll(true);
getCheckedNodesboolean(true)获取被选中的节点数组 (false)获取未被选中的节点数组
更多方法参见ztreeAPI
