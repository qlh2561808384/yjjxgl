初始化参数

名称类型描述默认值
widthintinput宽度100%
placeholderString提示文字""
requiredBoolean是否必选false
validateFunction OR String 定义表单验证规则。可使用globalValid中定义的正则名称 支持多种正则验证 多个正则名间用'|'进行分割。如正则无法满足 可定义function(value){return {result:false,msg:''}} 参数为表单value 返回值 包含boolean 标识验证是否成功 当result为false时 提示msg 错误信息false
validTextString自定义验证提示文本""
validFailFunction验证失败触发回调方法 参数为 fn jquery对象function($el){return false}
formatString日期格式， p, P, h, hh, i, ii, s, ss, d, dd, m, mm, M, MM, yy, yyyy 的任意组合"mm/dd/yyyy"
weekStartInteger一周从哪一天开始。0（星期日）到6（星期六）0
todayBtnBoolean, "linked"如果此值为true 或 "linked"，则在日期时间选择器组件的底部显示一个 "Today" 按钮用以选择当前日期。如果是true的话，"Today" 按钮仅仅将视图转到当天的日期，如果是"linked"，当天日期将会被选中。false
autocloseBoolean当选择一个日期之后是否立即关闭此日期时间选择器false
todayHighlightBoolean如果为true, 高亮当前日期。false
startViewNumber, String日期时间选择器打开之后首先显示的视图。 可接受的值：
0 or 'hour' for the hour view
1 or 'day' for the day view
2 or 'month' for month view (the default)
3 or 'year' for the 12-month overview
4 or 'decade' for the 10-year overview. Useful for date-of-birth datetimepickers.		4, 'year'
minViewNumber, String日期时间选择器所能够提供的最精确的时间选择视图0, 'hour'
forceParseBoolean当选择器关闭的时候，是否强制解析输入框中的值。也就是说，当用户在输入框中输入了不正确的日期，选择器将会尽量解析输入的值，并将解析后的正确值按照给定的格式format设置到输入框中。true
showMeridianBooleanThis option will enable meridian views for day and hour views.false
datetimepicker的用法具体可参考http://www.bootcss.com/p/bootstrap-datetimepicker/
